# Prerequisites

OS packages
- nvm
- pyenv
- ripgrep

Python packages
- aider-chat

Run `source ./bin/configure`

This will:
- Set up a Python virtual environment and install aider,
  which is used for code editing.
- Set the node version using nvm, and run `npm install`
- Install ripgrep
